# CO-AST-CO-Amine-Speciation-and-Titration
CO₂AST (CO₂–Amine Speciation and Titration) is a mechanistic modeling framework for simulating aqueous CO₂–amine systems. It predicts pH, solution speciation, and reaction thermodynamics by coupling carbonate chemistry, acid–base equilibria, carbamate formation, and titration or ITC experiments for parameter estimation and mechanistic analysis.
